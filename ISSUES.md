- The canonical current-state owners are `docs/LBA2_ZIG_PORT_PLAN.md`, `docs/codex_memory/current_focus.md`, and the relevant subsystem packs. Do not recreate a separate prompt-brief owner for the next slice.

Classic code uses raw HQR slot `0` for `SizeCube.MAX`, while the Zig tooling exposes physical entries starting at `1`, so `SCENE.HQR[2]` is classic loader scene `0` and `classic_loader_scene_number` must use `entry_index - 2` for real scenes.
